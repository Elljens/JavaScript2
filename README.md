<h2>Description</h2>
This website was created as a JavaScript 2 course assignment. The goal was to build a social media platform where you can view posts created by yourself and other users. The project focuses on working with APIs, allowing users to register, log in and view posts and profiles.

<h2>This project includes:</h2>
- a register page where you can create a new user<br>
- a log in page<br>
- a profile page where the logged in user can view their own profile and posts<br>
- pages that gives the registered user possibilities to create new, edit, or delete their own posts<br>
- a paginated index page where all posts from users are displayed<br>
- a search field to search for posts<br>
- a single post page where you can view details of a post based on its id<br>
- a user page with detail of other users, such as profile details and posts<br>
- an API client with reusable API services<br>

<h2>Built with</h2>
- HTML<br>
- CSS<br>
- JavaScript<br>
- ES5 modules<br>
